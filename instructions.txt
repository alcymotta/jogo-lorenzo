COMO ABRIR:
    1. Clique duplo em "index.html"
    2. O jogo abre automaticamente no navegador
    
NÃO PRECISA INSTALAR NADA!

==============================================================================

CONTROLES BASICOS:

PC:
  • W, A, S, D = Mover personagem
  • ESPAÇO = Pular
  • C = Abrir chat
  • Mouse Scroll = Zoom da câmera

MOBILE:
  • Joystick = Mover
  • Botão = Pular

==============================================================================

CODIGOS ESPECIAIS (Digite no chat):

65395  → Transformar em NUBE (personagem nuvem branca)
0762   → Transformar em c00lkidd (personagem rosa)  
77     → Transformar em CAPIVARA (personagem marrom)
888    → Transformar em GUARDA (personagem azul)
;fly   → ATIVAR VOO (levitar e voar livremente)

COMO USAR CODIGOS:

1. Pressione "C" para abrir o Chat
2. Digite o código (ex: 65395)
3. Pressione ENTER ou clique em "Enviar"
4. Pronto! Você se transformou!

MODO DEBUG (Para desenvolvedores):

Abra o Console (F12) e use:

GameDebug.teleport(0, 2, 0)     - Tele para coordenada
GameDebug.giveItem("hd_voador") - Dar item
GameDebug.transform("nube")     - Transformar
GameDebug.fly(true)             - Ativar vôo
GameDebug.playerInfo()          - Info do jogador
GameDebug.message("texto")      - Mensagem no chat

PROBLEMAS COMUNS E SOLUÇÕES:

PROBLEMA: Jogo não abre
SOLUÇÃO:  1) Tente outro navegador
          2) Verifique sua conexão de internet
          3) Teste com test.html para diagnosticar

PROBLEMA: Personagem não se move
SOLUÇÃO:  1) Clique dentro da janela do jogo
          2) Pressione WASD
          3) Tente F5 para recarregar

PROBLEMA: Códigos não funcionam
SOLUÇÃO:  1) Pressione "C" para focar no chat
          2) Digite o código exatamente (ex: 65395)
          3) Pressione ENTER ou clique "Enviar"

PROBLEMA: Muito lento/lag
SOLUÇÃO:  1) Feche outras abas/programas
          2) Tente reduzir a janela
          3) Mude para outro navegador
