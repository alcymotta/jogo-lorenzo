# ⚡ Início Rápido - Jogo Capibara 3D

## 1️⃣ Como Abrir o Jogo

### Forma Mais Fácil (Recomendada):
```
1. Abra a pasta "Jogo Lorenzo"
2. Clique duplo em "index.html"
3. Pronto! O jogo abre automaticamente
```

**Nenhuma instalação necessária!**

---

## 2️⃣ Primeiros Passos

```
1. Digite seu nome na tela inicial
2. Clique em "Entrar"
3. Use WASD para se mover
4. Espaço para pular
5. Aproveite!
```

---

## 3️⃣ Códigos Para Liberar Poderes

```
Ao abrir o jogo, pressione "C" para abrir o chat,
depois digite um dos códigos abaixo:

65395    → Virar Nube (personagem nuvem)
0762     → Virar c00lkidd (personagem rosa)
77       → Virar Capivara (personagem marrom)
888      → Virar Guarda (personagem azul)
;fly     → Ativar vôo
456      → Receber voador
```

---

## 4️⃣ Controles Rápidos

### PC:
```
W, A, S, D  → Mover
Espaço      → Pular
C           → Chat
Scroll      → Zoom câmera
```

### Mobile:
```
Joystick    → Mover
Botão       → Pular
```

---

## ❓ Não Abre?

**Solução 1:** Tente outro navegador (Chrome, Firefox, Edge)

**Solução 2:** Reinicie o computador (às vezes ajuda com WebGL)

**Solução 3:** Verifique internet (para carregar Three.js)

**Solução 4:** Se mesmo assim não funcionar, use teste.html para debug:
```
Clique duplo em "test.html" para ver o que está errado
```
