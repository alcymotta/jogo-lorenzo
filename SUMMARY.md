# 🏝️ Resumo Executivo - Ilha de Capibara 3D

## 📌 O Que É?

Um **jogo 3D interativo** completo que roda direto no navegador sem instalação. 
Exploração de uma ilha, transformações de personagem, voar, códigos especiais e muita diversão!

---

## ⚡ Começar em 3 Passos

```
1. Clique duplo em "index.html"
2. Digite seu nome
3. Divirta-se com WASD + Espaço!
```

**Pronto! Nada mais a fazer.** ✓

---

## 🎮 O Que Você Pode Fazer?

### Caminhar
- Use **WASD** para se mover
- **Espaço** para pular
- Explore toda a Ilha de Capibara

### Transformar
Digite código (pressione **C**):
- **65395** → Virar uma nuvem flutuante ☁️
- **0762** → Ficar rosa e chique 😎
- **77** → Virar uma capivara 🐹
- **888** → Virar um guarda azul 👮

### Voar
- **;fly** → Ativar modo vôo
- Pode se mover livremente por toda parte!

### Se Divertir
- **😂 Seu emote de riso** → Rir com expressão
- **🤸 Mortal** → Fazer pirueta girando
- Interagir com **capivaras** que andam pelo mapa

## 💻 Como Funciona Internamente

```

        ↓

        ↓
Three.js renderiza 3D
        ↓
GameEngine controla tudo
        ↓
Loop de animação (60fps)
        ↓
Você pressiona teclas
        ↓
Player se move
        ↓
Câmera segue
        ↓
Renderização atualiza
        ↓
Tela mostra nova imagem
```
