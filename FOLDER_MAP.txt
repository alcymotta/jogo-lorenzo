## 🎁 Bônus

### Modo Hidden Debug
Abra console (F12) e use:
```javascript
GameDebug.playerInfo()     // Suas stats
GameDebug.fly(true)        // Voar
GameDebug.teleport(0,0,0)  // Teleportar
GameDebug.transform()      // Transformar
```

3. Identifique o problema

- Ajustar velocidades
- Modificar configurações
